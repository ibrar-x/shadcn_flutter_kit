# Getting Started

`app` is a registry component in category `layout`.

Application wrapper for shadcn theme and overlay layers.

## Install

```bash
flutter_shadcn add app
```

## Import

```dart
import 'package:<your_app>/ui/shadcn/layout/app/app.dart';
```
