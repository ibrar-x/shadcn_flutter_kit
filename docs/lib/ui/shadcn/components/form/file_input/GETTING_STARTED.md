# Getting Started

`file_input` is a registry component in category `form`.

Utilities for mapping file extensions to icon widgets.

## Folder Map

- `README.md`: Primary component docs and usage guidance for contributors/users.
- `file_input.dart`: Widget/composite source file in this folder.
- `file_input.meta.json`: Generated readme/meta companion used by docs/index tooling.
- `meta.json`: Registry metadata used for manifests, dependencies, and tooling.
- `preview.dart`: Preview/demo entry used by gallery/docs environments.
- `_impl/`: Private implementation details that support `file_input`.
