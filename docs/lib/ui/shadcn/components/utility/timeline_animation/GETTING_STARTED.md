# Getting Started

`timeline_animation` is a registry component in category `utility`.

Keyframe-based animation timeline utilities.

## Install

```bash
flutter_shadcn add timeline_animation
```

## Import

```dart
import 'package:<your_app>/ui/shadcn/components/utility/timeline_animation/timeline_animation.dart';
```

## Complete Usage

Use the full example in `README.md` to run a complete app with `TimelineAnimation`, `AbsoluteKeyframe`, `RelativeKeyframe`, and `StillKeyframe`.

## Folder Map

- `README.md`: usage docs with complete runnable example.
- `meta.json`: registry metadata.
- `timeline_animation.dart`: public API for timeline/keyframe utilities.
- `timeline_animation.meta.json`: docs/index companion metadata.
- `_impl/`: internal timeline implementation details.
