# Avatar Group

Composite export that exposes AvatarGroup along with Avatar and AvatarBadge.

## Usage

```dart
import 'ui/shadcn/composites/avatar_group/avatar_group.dart';

AvatarGroup.toRight(
  children: const [
    Avatar(initials: 'AL'),
    Avatar(initials: 'BK'),
    Avatar(initials: 'CM'),
  ],
);
```
