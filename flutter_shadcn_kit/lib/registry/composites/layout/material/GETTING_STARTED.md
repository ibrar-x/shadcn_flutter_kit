# Getting Started

`material` is a registry composite in category `layout`.

Composite export for ShadcnUI, cards, buttons, and dialogs.

## Folder Map

- `README.md`: Primary component docs and usage guidance for contributors/users.
- `material.dart`: Widget/composite source file in this folder.
- `material.meta.json`: Generated readme/meta companion used by docs/index tooling.
- `meta.json`: Registry metadata used for manifests, dependencies, and tooling.
- `_impl/`: Private implementation details that support `material`.
