# Fonts (`shared/fonts`)

Bundled font assets for icon packs and typography presets.

---

## When to use

- Use this when:
  - you need the bundled icon fonts or Geist typography.
- Avoid when:
  - your app provides its own fonts.

---

## Contents

- `radix.otf` — Radix Icons font.
- `lucide.ttf` — Lucide Icons font.
- `bootstrap.otf` — Bootstrap Icons font.
- `geist/` — Geist and GeistMono font families.

---

## Notes

- Ensure your app bundles these fonts when using the icon sets.
