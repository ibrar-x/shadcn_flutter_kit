# Geist Fonts (`shared/fonts/geist`)

Geist and GeistMono font families with multiple weights and italics.

---

## When to use

- Use this when:
  - you want the default typography used by the registry.
- Avoid when:
  - your design system specifies a different typeface.

---

## Contents

- Geist weights from Thin to UltraBlack (with italics).
- GeistMono weights from Thin to UltraBlack (with italics).
