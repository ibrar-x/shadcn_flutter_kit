# Docs App Common Examples

Compatibility entry for the previous skill layout.

Use these rebuilt references:

- `forms.md` for form bundles, validation, date/time/color/file inputs.
- `composition.md` for shell and overlay composition.
- `theming-assets.md` for theme and asset workflows.

Default app-form install:

```bash
flutter_shadcn add \
  @shadcn/form \
  @shadcn/form_field \
  @shadcn/input \
  @shadcn/text_field \
  @shadcn/select \
  @shadcn/checkbox \
  @shadcn/radio_group \
  @shadcn/switch \
  @shadcn/slider
```
