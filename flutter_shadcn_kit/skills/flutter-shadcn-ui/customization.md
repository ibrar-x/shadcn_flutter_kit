# Customization Compatibility Reference

This file is kept for agents or scripts that still open `customization.md`.

Use `references/theming-assets.md` for the current theme, asset, and platform workflow.

```bash
flutter_shadcn theme --list
flutter_shadcn theme --apply modern-minimal
flutter_shadcn theme widget --list
flutter_shadcn assets --icons
flutter_shadcn assets --typography
flutter_shadcn platform --list
flutter_shadcn sync
flutter_shadcn validate --json
```
